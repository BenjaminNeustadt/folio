== Development/Configuration

To connect the constituent parts of the application.

Add with your API keys for the `.env` file.

.`.env`
[source, ini]
----
S3_ACCESS_KEY=''
S3_SECRET_KEY=''
MAPBOX_ACCESS_TOKEN=''
----
